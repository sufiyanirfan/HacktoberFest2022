## Technologies Used

- Python
- Framework - PyGame
- Other modules - time, os, sys, ctypes, etc.

## Guidelines to setup

### Prerequisites

- Python should be installed: `Python version > 3.6`
- Pygame should be installed: `Pygame version > 1.9.5`

### Run the game
```
python <file_path>/main.py
```

### Create a release
```
pyinstaller.exe .\main.spec --onefile --noconsole
```
